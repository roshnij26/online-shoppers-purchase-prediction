# online-shoppers-purchase-prediction
Machine learning model to predict whether an online visitor is likely to make a purchase based on their session behavior data.

## Dataset
- Source: Kaggle - online shoppers Dataset
- Shape: 12330 rows x 18 columns

## Technologies Used
- Python
- pandas
- NumPy
- scikit-learn
- Matplotlib
- Seaborn

## Workflow
1. Data Cleaning
2. Exploratory Data Analysis(EDA)
3. Feature Engineering
4. Data Preprocessing
5. Hyperparameter Tuning
6. Model Training
7. Evaluation

## Results
- F1 score: 0.634
